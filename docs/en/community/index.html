Community

The Polkadot community is expanding every day. Below are the most essential links to the community channels.

General

  • Join the Polkadot PoC-4 Testnet ("Alexander") - Access the UI dashboard for interacting with the latest Polkadot testnet.
  • Polkadot GitHub - Parity maintained repository that houses the Rust implementation of the Polkadot runtime environment.
  • Polkadot meetups - Information on hosting meetups, applying for funding, and materials for running it.

Riot

  • Polkadot Beginners Lounge - Room for beginners learning about Polkadot. A good place to ask questions
  • Polkadot Watercooler Chat - General room for talk about Polkadot.
  • Polkadot Validator Lounge - Room for validators learning about setting up a node.
  • Substrate Technical - More advanced room for technical questions on building with Substrate.

Socials

  • Subscribe to the Polkadot newsletter - Email sign up form is available at the bottom of the page.
  • Polkadot Telegram - Official Telegram room for discussion of Polkadot (please read the rules before chatting).
  • Polkadot Twitter - Official Twitter account of Polkadot.
  • Polkadot Reddit - Official Polkadot Reddit.
  • Polkadot Youtube Channel - Video content from founders and build of Polkadot describing the network and the vision.

Ecosystem

  • Teams Building on Polkadot - Maintained list of teams building on Polkadot and/or Substrate.
